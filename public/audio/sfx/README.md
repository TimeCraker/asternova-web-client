# UI / Button SFX Layout

Put WAV button and UI sound effects here.

Suggested structure:
- `public/audio/sfx/ui/` - generic UI sounds
- `public/audio/sfx/ui/buttons/` - button click / hover / confirm / back
- `public/audio/sfx/ui/sliders/` - drag / volume slider ticks
- `public/audio/sfx/ui/toggles/` - on/off switch effects
- `public/audio/sfx/gameplay/` - shared gameplay SFX across pages

Suggested file names:
- `click.wav`
- `hover.wav`
- `confirm.wav`
- `cancel.wav`
- `back.wav`
